Campus Map App
Sivas Cumhuriyet Üniversitesi kampüs yaşamını dijitalleştirmek ve kullanıcıların yerleşke içinde yollarını kolayca bulmalarını sağlamak amacıyla geliştirilmiş bir mobil navigasyon uygulamasıdır.
Özellikler:

* İnteraktif Kampüs Haritası: Fakülteler, sosyal tesisler ve idari birimlerin detaylı yerleşimi.
* Kategori Bazlı Arama: Aranan binaya veya birime hızlı erişim.
* Modern Arayüz: Kullanıcı dostu ve hızlı bir mobil deneyim.
  
Kullanılan Teknolojiler:

* Framework: Flutter 
* Dil: Dart 
* Versiyon Kontrol: Git & GitHub

Kurulum

Projeyi yerel cihazınızda çalıştırmak için:

1.Depoyu forklayın veya klonlayın.

2.Terminalde flutter pub get komutunu çalıştırın.

3.flutter run ile uygulamayı başlatın.
